Gobo Eiffel Cop

Gobo Eiffel Cop, or gecop for short, is an open source tool to validate the conformance of Eiffel tools (typically compilers) to the ECMA Eiffel standard. It also tests the interoperability with the ISE Eiffel compiler. It comes with a validation suite to exercise the Eiffel tool under test.
