There is another tool which is able to generate Eiffel classes from image files. It's called Eiffel Image Embedder Tool and it is included in ISE EiffelStudio's package in $ISE_EIFFEL/tools/spec/$ISE_PLATFORM/bin/eimgemb. See Implementation of geimage to see why geimage has been introduced despite the existence of this other tool.
