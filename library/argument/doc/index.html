Gobo Eiffel Argument Library

The Gobo Eiffel Argument Library is a library to parse command line arguments of Eiffel applications.

This document is structured as follows:

Introduction
A short guide into the argument library and an example with explanations.
Parser
A guide to the interface of the argument parser and a description of important features.
Options
A reference of available options and a guide on how to create customized option types.
Lists of alternative options
Lists of alternative options are an advanced way to specify options sets that are used for different operation modes of the application.
Downloading Instructions
License agreement and downloading instructions.
