Supported Eiffel Compilers

The classes provided in the Gobo Eiffel package should compile and work without any change on the currently maintained Eiffel compilers (see Readme.txt and Release_notes.txt). In order to make things easier, only the latest public release of each compiler will be supported. This latest release can be a beta version provided that it is deemed stable enough.

The currently supported Eiffel compilers are:

  • See Readme.txt and Release_notes.txt.

Developers should use these compilers when testing their libraries.
