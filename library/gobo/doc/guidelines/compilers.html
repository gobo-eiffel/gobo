Supported Eiffel Compilers

The classes provided in the Gobo Eiffel package should compile and work without any change on the currently supported Eiffel compilers. These compilers are those which are as close as possible to the ECMA standard and the FreeELKS kernel library. In order to make things easier, only the latest public release of each compiler will be supported. This latest release can be a beta version provided that it is deemed stable enough.

The currently supported Eiffel compilers are:

  • See Eiffel compilers and Release_notes.md.

Developers should use these compilers when testing their libraries.
