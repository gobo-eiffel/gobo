Gobo Eiffel String Formatter Library

The Gobo Eiffel String Formatter Library gives Eiffel programmers access to the functionality of the Standard C library sprintf. The Gobo Eiffel String Formatter Library is fully implemented in Eiffel. It does not use the C function.
